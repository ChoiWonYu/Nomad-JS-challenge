CSS 사용해서 스타일 바꾸기

1.className을 바꾸는 방법

classList 사용
classList.contain(element ) : element로 전달 되는 className이 있는지 확인해줌
classList.add( element) : element로 전달 되는 className을 추가해줌
classList.remove(element ) : element로 전달 되는 className을 삭제해줌
->toggle

classList.toggle( element ) : element로 전달 되는 className을
있는지 확인하고 없으면 추가, 있으면 삭제 해준다.
->위의 작업을 한번에 해줄 수 있다.
